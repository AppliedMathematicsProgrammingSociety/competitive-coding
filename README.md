## Learning Path

### 1. Basics (Week 1-2)
- **Time Complexity Analysis**
  - Big O Notation
  - Best, Average, and Worst Cases
  - Common Time Complexities

- **Space Complexity**
  - Memory Usage Analysis
  - Trade-offs between Time and Space

- **Problem-Solving Patterns**
  - Multiple Pointers
  - Sliding Window
  - Frequency Counter
  - Divide and Conquer

### 2. Data Structures (Week 3-6)

#### Basic Data Structures
- **Arrays and Strings**
  - Array Operations
  - String Manipulation
  - Multi-dimensional Arrays
  
- **Linked Lists**
  - Singly Linked Lists
  - Doubly Linked Lists
  - Circular Linked Lists

- **Stacks and Queues**
  - Stack Implementation
  - Queue Implementation
  - Priority Queues
  - Deques

#### Advanced Data Structures
- **Hash Tables**
  - Hash Functions
  - Collision Resolution
  - Implementation

- **Trees**
  - Binary Trees
  - Binary Search Trees
  - Tree Traversals
  - Balanced Trees (AVL, Red-Black)

- **Graphs**
  - Graph Representation
  - Graph Traversals
  - Basic Graph Algorithms

### 3. Algorithms (Week 7-12)

#### Searching and Sorting
- **Searching Algorithms**
  - Linear Search
  - Binary Search
  - Jump Search

- **Sorting Algorithms**
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort

#### Advanced Algorithms
- **Recursion**
  - Basic Recursion Problems
  - Backtracking
  - Divide and Conquer

- **Dynamic Programming**
  - Memoization
  - Tabulation
  - Common DP Patterns

4. Solution explanations
